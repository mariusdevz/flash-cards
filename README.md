# Flash Cards App

https://roadmap.sh/projects/flash-cards
A simple Flash Cards application built with React that allows users to review questions and answers interactively.

## Features

- Display one flash card at a time
- Show and hide answers
- Navigate between cards
- Progress tracking
- Dynamic progress bar
- Responsive layout

## Technologies Used

- React
- JavaScript (ES6+)
- CSS3

## Learning Objectives

This project helps practice:

- React State Management
- Component-Based Architecture
- Event Handling
- Conditional Rendering
- Array Data Management
- UI State Updates

## Project Structure

```
src/
├── App.jsx
├── App.css
└── main.jsx
```

## How It Works

1. A question is displayed.
2. Clicking "Show Answer" reveals the answer.
3. Clicking "Hide Answer" hides it.
4. Previous and Next buttons navigate through cards.
5. The progress bar updates automatically.

## Getting Started

```bash
npm install
npm run dev
```
